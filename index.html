PureDogPWA

An installable Progressive Web App (PWA) responsive website that looks good on desktop, tablet, or mobile. It just works.

Take this as a template it, customize it as your own.

PureDogPWA is a mash up of three things

  1. Pure CSS responsive template at purecss.io. This means it resizes and simply works regardless of the screen size, device, or browser.
  2. Doggone Minimal Website (DMW) at github.com/crrdlx/doggone-minimal-website. This means it always looks good and loads fast.
  3. SimplePWA template at simplepwa.com. This means it can be installed on your device as a Progressive Web App. That's good because (1) it takes very little storage space on your device, (2) it looks, acts, and feels like a regular app, (3) updates are instantaneous, and (4) you don't need to rely on an app store.

How to use this layout

To use this layout, you can just copy paste the HTML, along with the CSS in side-menu.css, and the JavaScript in ui.js. The JS file uses vanilla JavaScript to simply toggle an active class that makes the menu responsive.
