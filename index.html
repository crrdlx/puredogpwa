PureDogPWA

An installable Progressive Web App (PWA) responsive website that looks good on desktop, tablet, or mobile. It just works.

Take this as a template it, customize it as your own.

PureDogPWA is a mash up of three things

  1. Pure CSS responsive template at PureCSS.io. This means it resizes and simply works regardless of the screen size, device, or browser.
  2. Doggone Minimal Website (DMW) at github.com/crrdlx/doggone-minimal-website. This means it always looks good and loads fast.
  3. SimplePWA template at SimplePWA.com. This means it can be installed on your device as a Progressive Web App (PWA). That's good.

How to use this website template

Grab the code from github.com/crrdlx/doggone-minimal-website. You can either...

  • Clone the repository from GitHub above (choose "Code", then "Clone"), or
  • Download the zip folder with all the files in it (choose "Code", then "Download ZIP").
